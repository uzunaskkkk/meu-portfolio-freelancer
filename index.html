Como a Mágica Acontece:

O formulário ao lado é feito com HTML e JavaScript. Quando você clica em enviar, o JavaScript valida os dados e (em um projeto real) os envia para um "backend".

Abaixo está um exemplo de como um backend em Python, usando o framework Flask, receberia sua mensagem:

# backend.py - Um exemplo simples com Flask (Python)

from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/submit_form', methods=['POST'])
def handle_form():
    data = request.get_json()
    
    name = data.get('name')
    email = data.get('email')
    message = data.get('message')

    # Aqui, o código enviaria um e-mail, salvaria no banco de dados, etc.
    print(f"Nova mensagem de {name} ({email}): {message}")

    # Retorna uma resposta de sucesso para o frontend
    return jsonify({'status': 'success', 'message': 'Mensagem recebida!'})

if __name__ == '__main__':
    app.run(debug=True)
